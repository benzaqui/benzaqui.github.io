Ma Voix

iPhone · Communication · Voix

What this app is

Ma Voix is an iPhone app for people who have difficulty speaking (for example because of a neurological condition). They compose or choose short French phrases; the app reads them aloud using the device voice or optional high-quality cloud voices. The goal is everyday conversation with family, caregivers, and friends—quickly and clearly.

Main features

  • Text + speak: type or pick text, then play it with one action.
  • Suggestions: on-device and optional cloud-assisted phrase ideas.
  • Favorites: save phrases you use often.
  • People: optional hints about frequent names and context (stored on the device).
  • Urgency / quick access: shortcuts for important messages when needed.

Important notes

  • The user interface is French-oriented.
  • There is no user account inside the app; nothing is “logging you in” to Ma Voix.
  • Optional cloud features send only what is needed for that feature (e.g. the phrase to synthesize or to suggest completions).

En bref (français)

Ma Voix aide une personne qui ne peut pas parler aisément à faire entendre des phrases en français (saisie ou choix rapides), avec la voix du téléphone ou une voix cloud optionnelle. Favoris, suggestions, raccourcis : le tout pensé pour le quotidien avec les proches et les soignants.
